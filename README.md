# Alpine WCLI nginx php7

What does this image contain?

- Alpine:latest
- nginx:latest
- WPCLI:latest
- php7.0x and php7.0x-fpm

## Contributing

Run: 
```$bash
npm install
```
Make sure the lint-ers pass.
